# Frontend

Le frontend Essensys est une application React moderne.

## Caractéristiques

- **Framework** : React + TypeScript
- **Build tool** : Vite
- **Emplacement** : `/opt/essensys/frontend/dist/`
- **Servi par** : Nginx (local) et Traefik (WAN)

## Fonctionnalités

- Interface utilisateur moderne
- Configuration dynamique du backend
- Détection automatique du domaine WAN
- Compatible avec les navigateurs modernes

## Build

Le frontend est compilé en fichiers statiques HTML/CSS/JS et servi par Nginx.
